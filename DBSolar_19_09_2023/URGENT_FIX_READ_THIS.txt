================================================================================
WHY THIS IS NEEDED
================================================================================

The database columns don't have DEFAULT values set. When Django tries to insert
a new row, PostgreSQL rejects it because:
- The id column is NOT NULL
- There's no DEFAULT value
- PostgreSQL can't generate an ID automatically

The fix script sets: DEFAULT nextval('sequence_name') which tells PostgreSQL
to automatically generate IDs.
